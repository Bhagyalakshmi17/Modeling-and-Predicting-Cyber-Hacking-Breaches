Cybersecurity Breach Analysis & Prediction

This project analyzes cyber hacking breach incidents spanning 12 years to identify trends, patterns, and factors influencing cybersecurity breaches.

Objectives
Analyze breach frequency and severity trends.
Perform statistical analysis on cyber incident data.
Explore predictive approaches for cybersecurity risk assessment.
Study breach inter-arrival times and breach sizes.
Technologies
Python
Machine Learning
Support Vector Machine (SVM)
MySQL
Django
Key Findings
Cyber hacking incidents increased in frequency over time.
Statistical and predictive models were used to analyze breach patterns.
Data visualization and trend analysis provided insights into cybersecurity risks.
