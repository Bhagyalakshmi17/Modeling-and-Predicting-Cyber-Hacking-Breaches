Modeling and Predicting Cyber Hacking Breaches
Overview

This project focuses on analyzing and predicting cyber hacking breach incidents using statistical analysis and machine learning techniques. The study examines cybersecurity breach data collected over a 12-year period to identify trends, patterns, and factors influencing cyber-attacks.

Objectives
Analyze cybersecurity breach frequency and severity.
Identify patterns and trends in cyber hacking incidents.
Apply machine learning techniques for predictive analysis.
Evaluate breach occurrence and potential impact.
Generate insights that support cybersecurity risk assessment and decision-making.

Methodology
Collected and analyzed cyber hacking breach incident data.
Performed statistical analysis to identify breach trends and patterns.
Applied Support Vector Machine (SVM) for predictive modeling.
Evaluated breach frequency, severity, and risk factors.
Generated visualizations and insights to support cybersecurity analysis.
Key Findings
Cyber hacking incidents increased in frequency over time.
Statistical and predictive models can be used to analyze cybersecurity risks.
Data-driven approaches help identify patterns in cyber breach activities.
Predictive analytics can support proactive cybersecurity planning and risk mitigation.
